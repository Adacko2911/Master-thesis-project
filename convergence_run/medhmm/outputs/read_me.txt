In this folder all outputs generated by the '.\convergence_run\medhmm\convergence_run_medhmm.R' and 
'.\convergence_run\medhmm\convergence_run_medhmm2.R'

### Output files with models data 
Those files will have the following name structure: 
convergence_run_m <number of states> _nt <observation scenario> _dwell <dwell time scenario> dt_set <chosen dataset identifier> _it4000_burn_in2000.rds

The outputs files include a five element list: 
	--> 1. chain1: first chain model output 
	--> 2. MAP1: the maximum a posteriori estimates for chain 1
	--> 3. chain2: second chain model output
	--> 4. MAP2: the maximum a posteriori estimates for chain 2
	--> 5. gelman_rubin_stats: estimates of the gelman rubin scale reduction statistics for group level parameters

### The RDS data files with all gelman rubin statistics combined named :
all_gelman_medhmm_m3.rds
all_gelman_mhmm_m4.rds

### The pdf with all traceplots that is generated by the '.\convergence_run\medhmm\convergence_run_mhmm.R' script named: trace_plots_medhmm.pdf
